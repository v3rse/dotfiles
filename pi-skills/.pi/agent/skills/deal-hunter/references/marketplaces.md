## Search Tips by Marketplace

### eBay.de
- Use filters: "Kaufen", "Sofort-Kaufen", "Neuware", "Gebraucht"
- Sort by price + shipping
- Check seller feedback: 95%+ positive, 100+ sales for unknown sellers
- "eBay Garantie" listings are safer

### KleinAnzeigen
- Filter: "Direkt kaufen" (Buy It Now) — avoid "VB" (negotiable) unless you want to haggle
- Filter: "Versand möglich" — unless you want local pickup only
- Prefer "gewerblich" (commercial) sellers over private
- Read description for hidden defects: "Funktioniert einwandfrei" = works perfectly; "Defekt" = broken
- Ask for photos of actual item, not stock photos

### Refurbished Shops
- Check condition grades: "Wie neu" > "Sehr gut" > "Gut" > "Akzeptabel" > "Fair"
- "Wie neu" often means open-box returns, excellent value
- Compare warranty length — some offer up to 36 months
- Check if original box/accessories included

### Amazon Warehouse
- Search product, then filter "Amazon Warehouse"
- Conditions: "Wie neu" = like new; "Sehr gut" = very good; "Gut" = good
- Prime shipping, 30-day return policy
- Often the safest "used" option

## Country-Specific Notes

**Default: Germany**
- All prices in EUR
- Shipping within DE: typically €0–15
- 14-day return right for online purchases (EU consumer law)
- 2-year warranty on new items (EU law)

**Austria:**
- Most German sellers ship to AT for €5–15
- Local: willhaben.at (Austrian KleinAnzeigen equivalent)

**Switzerland:**
- German sellers often don't ship to CH due to customs
- Check ricardo.ch, tutti.ch, anibis.ch for Swiss marketplaces
- Factor in 7.7% VAT + customs processing fees

**Other EU:**
- Check seller's shipping policy before counting a deal
- Customs/VAT may apply for non-EU sellers
