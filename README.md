dotfiles
========

My dotfiles

Features
--------
Nothing much...just my own dotfiles and oh-my-zsh.

Installation
------------

Install them  by running:

```
    git clone https://github.com/v3rse/dotfiles.git ~/dotfiles
    chmod +x ~/dotfiles/bin/install.sh
    sh install.sh
```
